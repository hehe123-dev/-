# 专利智能评估系统 - 后台管理模块

## 功能概述

系统新增了完整的后台管理功能，管理员可以通过后台管理界面进行用户管理和评估指标权重配置。

## 主要功能

### 1. 登录与权限控制
- **管理员账号** (admin / admin123)：登录后跳转到后台管理系统
- **评估员账号** (evaluator / 123456)：登录后跳转到文档上传页面
- 支持"记住我"功能
- 页面展示测试账号，一键快速登录

### 2. 用户管理（admin.html）
- 查看所有账号列表（序号、账号、姓名、角色、创建时间）
- 添加新账号（账号、姓名、密码、角色）
- 编辑现有账号信息
- 删除账号（admin系统账号不可删除）
- 数据存储在 localStorage

### 3. 评估指标权重配置（admin.html）
按产业类型独立配置权重：
- **战略性新兴产业**
- **未来产业**  
- **传统产业**

**一级指标：**
- 法律价值
- 技术价值
- 经济价值

**二级指标：**
- 法律价值：新颖性、创造性、实用性、稳定性
- 技术价值：技术先进性、技术复杂度、技术成熟度、可替代性
- 经济价值：市场前景、产业化周期、经济效益、竞争优势

每个指标通过滑块调整权重（0-100%），配置保存到 localStorage

## 使用流程

1. 打开 `login.html`
2. 使用管理员账号登录（或点击快速登录按钮）
3. 自动跳转到后台管理页面
4. 左侧菜单切换"用户管理"和"评估指标权重"
5. 进行相应的配置操作
6. 点击"返回首页"可跳转到文档上传页面

## 技术说明

- 前端框架：纯 HTML/CSS/JavaScript
- 数据存储：localStorage（演示用）
- 权限控制：基于角色的访问控制（RBAC）
- 响应式设计：适配不同屏幕尺寸

## 注意事项

⚠️ 当前为演示版本，所有数据存储在浏览器本地
⚠️ 生产环境需要：
  - 后端 API 接口
  - 数据库持久化
  - 安全加密传输
  - Token 认证机制
